INTRO TO GAME DESIGN

These games are made using p5.js, a JavaScript library for creative coding. Group projects are made through multiple prototyping and game testings to optimize gaming experience. Some of the games are developed and hosted on Glitch to make collaboration easier.

Tic Tac Toe

Controls
Arrow Keys: Move
Space: Place X/O/Tri

Instruction
3 players take turns to place their elements, black tile is not placeable.
Press "r" to restart.

Win condition: 3 in a row to win
(click inside the game screen before you move)

Snake

Controls
Arrow Keys: Change direction
Space: Pass through everything for 6 seconds

Powerups
Speed Reduction: Slows down temporarily
Reverse Growing: Grows shorter for next 5 food

Instruction
Score > 15: food will shift
Score > 25: spawn obstacles
Score > 35: spawn laserbeams
Press "r" to restart.

Sauce Is The Boss

A tower defense game with a twist, where you have to throw pasta to fight against the mafia.

This project is a team effort, and is made possible using Gltich that hosted the game.

Llama Dungeon Escape

2D dungeon crawler - You are a llama who takes on an adventure to save the farmer's chickens from the evil foxes.

This project is a team effort, and is made possible using Gltich that hosted the game.
